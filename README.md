# Визуализатор топологической сортировки

## Возможности

- Загрузка графа из текстового файла и создание графа мышью прямо на холсте
  (клик - вершина, клики по двум вершинам - ребро от первой ко второй,
  перетаскивание - перенос, правый клик - удаление).
- Пошаговое выполнение алгоритма с текстовым пояснением каждого шага:
  одна вершина обрабатывается за два шага (взятие из очереди готовых и
  удаление исходящих рёбер).
- Шаг вперёд, шаг назад, автопрогон с настраиваемым интервалом (100..2000 мс),
  сброс.
- Подсветка: у необработанных вершин подписана текущая входящая степень,
  вершины в очереди - жёлтые, обработанные - зелёные с позицией в порядке,
  текущая - красный контур, удалённые рёбра - пунктир.
- Панель состояния: очередь готовых вершин и построенный порядок.
- Обнаружение циклов: если сортировка невозможна, программа сообщает об этом
  и подсвечивает вершины, застрявшие в цикле.
- Сохранение результата в файл. Файл результата содержит и сам граф,
  поэтому его можно открыть обратно в программе - он сразу отобразится
  в завершённом виде, с готовым порядком и подсветкой.
- Обработка некорректных входных файлов: программа не падает, ошибка
  показывается с номером строки.
- Кнопка "О разработчиках".

## Формат входного файла

Текстовый файл, кодировка UTF-8. Пустые строки и строки, начинающиеся
с #, игнорируются.

```
V <номер> <x> <y>     вершина: номер от 1, координаты 0..1000
E <из> <в>            ориентированное ребро
```

Пример:

```
V 1 100 150
V 2 420 120
V 3 260 480
E 1 2
E 1 3
E 3 2
```

Ограничения: до 50 вершин, до 300 рёбер, петли и повторные рёбра
запрещены.

## Файл результата

Кнопка "Сохранить результат" записывает итог сортировки (комментариями #)
и сам граф в формате V/E, причём вершины идут в топологическом порядке.
Такой файл можно открыть через "Открыть" - программа распознает его
и сразу покажет завершённое выполнение: порядок в панели справа, зелёные
вершины с позициями, итог в журнале. Дальше можно разобрать выполнение
кнопкой "Шаг назад" или начать заново через "Сброс".

## Сборка и запуск

Требуется JDK не выше 21.

```
javac --release 21 -encoding UTF-8 -d out $(find src -name "*.java")
java -cp out toposort.App
```

Сборка исполнимого jar:

```
echo "Main-Class: toposort.App" > manifest.txt
jar cfm TopsortVisualizer.jar manifest.txt -C out toposort
java -jar TopsortVisualizer.jar
```

## Структура проекта

```
src/toposort/
    App.java              точка входа
    model/                граф и алгоритм (не зависит от интерфейса)
        Vertex.java       вершина (номер, координаты)
        Edge.java         ориентированное ребро
        Graph.java        граф, проверка ограничений, редактирование
        TopoSort.java     алгоритм Кана, полный прогон
        TopoStepper.java  пошаговое выполнение (для визуализации)
    io/
        GraphIO.java      чтение графа из файла, запись результата
    ui/
        MainFrame.java    главное окно, кнопки, таймер автопрогона
        GraphPanel.java   холст: отрисовка графа и редактирование мышью
examples/                 примеры входных файлов
```
